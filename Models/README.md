# Baseline Models for Surprise3D

We modified the code from [Intent3D](https://github.com/WeitaiKang/Intent3D) and [Reason3D](https://github.com/KuanchihHuang/Reason3D) to support training and evaluation on the Surprise3D dataset. We thank their dedicated efforts on these impressive works.

## Project Page

[liziwennba.github.io/SURPRISE3D](https://liziwennba.github.io/SURPRISE3D)

## Available Models

| Model | Directory | Original Paper | Task |
|:--|:--|:--|:--|
| **Reason3D** | [reason3d/](reason3d/) | [3DV 2025](https://arxiv.org/abs/2405.17427) | 3D reasoning segmentation |
| **Intent3D** | [intent3d/](intent3d/) | [ICLR 2025](https://arxiv.org/abs/2405.18295) | Intention-based 3D grounding |

Please refer to each model's README for detailed installation, data preparation, training, and evaluation instructions.
